# Mutual Fund Facts Bot

This project is organized into exactly three application folders:

- `backend` - FastAPI backend with MVC-style structure and the RAG pipeline
- `frontend` - React + Vite search-first UI intended for Vercel
- `doc` - architecture, deployment, and quickstart documentation

The bot answers factual scheme and statement-help questions from official AMC, AMFI, and SEBI sources. It does not provide investment advice.
